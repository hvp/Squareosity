Squareosity
===========

2D retro neon super awesome physics puzzler. 

Controls
========

Xbox 360 GamePad:
- Left Thumbstick = Movement
- Right Thumbstick = Laser
- A = Respawn 
- Back / Start = Pasue


PC:
- Arrow keys = Movement 
- Mouse + LMB = Laser 
- A = Respawn
- ESC = Pasue

Rules
=====

- You can only touch blue and red neon objects.
- You must collect stars

Build requirements
==================
- Visual C# 2010
- XNA 4.0
